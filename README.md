A full-stack Netflix-style web app built with React, Node.js, and MongoDB.

## Features
- User authentication (JWT/Firebase)
- Browse movies via TMDB API
- Add/remove movies to watchlist
- Responsive UI

## Tech Stack
- React.js
- Node.js + Express
- MongoDB
- TMDB API

## Setup
1. Clone repo
2. Run `npm install` in both frontend & backend
3. Add TMDB API key in `.env`
4. Run backend: `npm start`
5. Run frontend: `npm start`
